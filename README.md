## MathRobot :

Sa mission : résoudre le théorème de Pythagore, le célèbre mathématicien ainsi que les fonctions trigonométriques. 

J'espère que ce MathRobot vous plaira!

## Mode d'emploi :

Ce dossier contient deux programmes : un qui s'éxécute dans un terminal classqiue (genre celui de VSCode, après avoir bien sur téléchargé l'appli et python sur python.org) et un qui s'éxécute avec Tkinter, un module qu'il faut installer via, en fonction de votre version de pip : pip3 install tkinter ou pip install tkinter. Ensuite, dans les deux cas, sur un éxécuteur comme VSCode, après avoir installé Python, lancez le fichier et laisser PythRobot ou Trigobot vous guider!
